Detailed Description

struct worker - A worker for processing received packets.

Each worker is launched on a slave lcore and currently handles ONLY one RX and TX queue of a port. E.g. for 2 ports with 2 queues per port, worker 0 handle rx/tx queue 0 of port 0 and 1, and worker 1 handle rx/tx queue 1 of port 0 and 1. This dispatch focuses on performance, multiple queues are handled by different lcores even for single port
